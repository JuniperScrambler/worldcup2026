# 選手顔写真の追加・反映手順書

このディレクトリに選手の顔写真（画像ファイル）を配置することで、アプリ内の「選手一覧」「スタメン選出」「サッカーピッチ」すべての箇所に本物の顔写真が反映されます。

---

## 📸 反映手順

### ステップ 1: 写真ファイルを用意する
各選手の顔写真（画像）を用意します。

### ステップ 2: ファイル名を「背番号.jpg」に変更する
アプリ側で背番号に対応して自動的に読み込むため、ファイル名を以下のように**背番号の半角数字**に変更し、拡張子を **`.jpg`** にします。

*   **鈴木 彩艶 選手**: `1.jpg`
*   **冨安 健洋 選手**: `16.jpg`
*   **久保 建英 選手**: `7.jpg`
*   **上田 綺世 選手**: `9.jpg`

> **※注意**:
> - 拡張子は小文字の `.jpg` に統一してください（`.JPG` や `.jpeg`, `.png` は読み込まれません）。
> - トリミングは、アプリ内で自動的に「正方形」かつ「丸型」に切り抜かれますので、基本的にはアスペクト比 1:1（正方形）に近い、顔が中央に写っている写真がおすすめです。

### ステップ 3: このフォルダ（assets/players/）に配置する
画像ファイルをこのフォルダ（`/assets/players/`）の直下に保存します。

## 🔄 反映の確認方法
画像を保存した後、iPhoneのSafariでアプリを開き、**「画面を下に引っ張って更新（プルダウン）」**するか、ページをリロードしてください。自動的にイラストから本物の顔写真に切り替わります。
